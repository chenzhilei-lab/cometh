## 🪐 FFP-pipeline — JWST Free-Floating Planet Pipeline

**→ [`/FFP-pipeline/`](FFP-pipeline/)**

Reproducible analysis pipeline for the manuscript *"Phantom multi-band free-floating planet candidates from archival JWST pipelines"* (submitted to RAA, 2026). Demonstrates how two systematic errors in archival NIRCam pipelines manufacture spurious FFP candidates in crowded star-forming regions:
1. A fixed 1.55 mag cross-detector colour bias from omitting pixel solid-angle conversion
2. Poisson neighbour misassignment at rate P = 1 − exp(−ρπr²)
